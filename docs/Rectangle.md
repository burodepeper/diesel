# Rectangle

## Usage

```coffeescript
square = new Square()
square.setPosition(10, 10)
square.setSize(40)
square.setColor(new Color('#e10'))
square.setOpacity(0.5)

rectangle = new Rectangle()
rectangle.setPosition(5, 5)
rectangle.setSize(50, 25)
rectangle.setColor(new Color('#5d0'))
rectangle.setOpacity(0.5)
```
